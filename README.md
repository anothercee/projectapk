# EcoLife Buddy (Web) - Demo

Sebuah web-app sederhana (HTML/CSS/JS) untuk prototipe EcoLife Buddy:
Utility (habit tracker, carbon calculator), Edukasi (fakta & tips), dan Fun (quiz, badge).

## Cara menjalankan lokal
1. Clone / download repo
2. Buka `index.html` di browser (double-click)  
   - Atau gunakan Live Server extension di VSCode untuk reload otomatis.

## Cara push ke GitHub
1. `git init`  
2. `git add .`  
3. `git commit -m "Initial web app EcoLife Buddy"`  
4. Buat repo di GitHub -> `git remote add origin <URL>`  
5. `git push -u origin main`

## Deploy ke GitHub Pages
1. Push repo ke GitHub.
2. Di GitHub repo -> Settings -> Pages -> Source: `main` branch, folder `/ (root)` -> Save.
3. Tunggu beberapa menit, lalu buka `https://<username>.github.io/<repo>`.
